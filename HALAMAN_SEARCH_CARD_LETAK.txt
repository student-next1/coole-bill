2️⃣ USER KLIK "🆔 Kartu ID" ← TRIGGER
   ════════════════════════════════════
   
   JavaScript function dipanggil:
   selectPaymentMethod('kartu_id')
   
   ↓
   
   POST request ke: /transaksi/select-payment
   (Payload: items, subtotal, total, method='kartu_id')
   
   ↓
   
   Server processor:
   - Simpan items, subtotal, total ke SESSION
   - Cek metode = 'kartu_id'
   - Redirect ke: /transaksi/search-card
   
   ↓ ↓ ↓ HALAMAN BERUBAH ↓ ↓ ↓

4️⃣ USER KLIK "✓ Gunakan Kartu Ini"
   ═════════════════════════════════════
   
   JavaScript function dipanggil:
   selectCard(cardId)
   
   ↓
   
   POST request ke: /transaksi/select-card
   (Payload: card_id=1)
   
   ↓
   
   Server processor:
   - Simpan payment_card_id ke SESSION
   - Redirect ke: /transaksi/invoice?method=kartu_id
   
   ↓ ↓ ↓ HALAMAN BERUBAH LAGI ↓ ↓ ↓
